Advanced Machine Learning & Forecasting — Python

End-to-end ML workflows combining supervised & unsupervised learning with time-series forecasting. Focus on clean feature pipelines, model selection/tuning, interpretable outputs, and decision-ready analytics.

Datasets: Tabular (classification & regression) and time-series (forecasting).

Models:

  • Supervised: Linear/Logistic Regression (Ridge/Lasso/Elastic Net), Decision Trees, Random Forests, Gradient Boosting, SVM, k-NN
  • Unsupervised: K-Means, Hierarchical (Agglomerative), DBSCAN; PCA for dimensionality reduction
  • Forecasting: ARIMA/SARIMA, Exponential Smoothing; ARCH/GARCH for volatility

Analyses / Experiments:

  • Data cleaning, missing-value handling, outlier treatment, encoding & scaling
  • Pipelines & ColumnTransformer (leak-free train/val/test)
  • Hyperparameter tuning (GridSearchCV / RandomizedSearchCV)
  • Clustering & segment profiling (elbow, silhouette)
  • Dimensionality reduction (variance explained, 2D/3D visuals)
  • Time-series diagnostics (stationarity, ACF/PACF, STL) & multi-horizon forecasts
  • Anomaly detection (residuals/thresholding, cluster distance)
  • Model interpretation (feature importance, permutation; optional SHAP)

Metrics:

  • Classification: Accuracy, Precision/Recall, F1, ROC-AUC, PR-AUC, Confusion Matrix
  • Regression: RMSE, MAE, R², residual diagnostics
  • Forecasting: MAE, RMSE, MAPE, AIC/BIC, residual whiteness tests

Stack: Python, pandas, NumPy, scikit-learn, statsmodels, matplotlib, seaborn.
